# 방콕 캠핑

앱인토스 웹뷰용 힐링 방치형 미니게임 프로토타입입니다.

## 실행

```bash
npm run dev
```

## 빌드

```bash
npm run build
```

게임 본문은 `public/game.html`에 단일 HTML/CSS/JavaScript로 들어 있습니다.
미확정 항목은 `CHECKLIST.md`에 정리했습니다.

Apps in Toss 프로젝트입니다.

## 시작하기

```bash
npm run dev
```
